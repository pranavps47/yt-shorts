# Story #1 Production Guide

**Title:** The Apartment Has One Extra Room That Wasn't There Yesterday
**Total narration length:** 3:22 (target final video: 3:25–3:35 with intro tail + outro)
**Aspect ratio:** 16:9 horizontal (1920×1080)
**Voice:** en-US-GuyNeural (Microsoft Edge Neural TTS, -8% rate)
**Files generated:**
- `story1_narration.mp3` — the narration audio (drag onto Audio Track 1)
- `story1_narration.srt` — subtitles (drag onto CapCut for auto-import, or use auto-captions instead)
- `story1_script.txt` — raw script text

## Step-by-step in iMovie

1. **Open iMovie** (Launchpad → iMovie) **→ Create New → Movie → No Theme**
2. **Project Settings** (gear icon, top right of timeline) → set Aspect Ratio to **Widescreen (16:9)**
3. **Import the narration:** File → Import Media → select `story1_narration.mp3` → drag it onto the timeline. It becomes Audio Track 1 (the purple bar). This anchors the whole video at 3:22.
4. **Download 12 B-roll clips** from the Pexels links below (one per scene). Hit "Free Download" → 1080p HD. Save them all in a single folder, e.g. `~/Downloads/yt_broll/`
5. **Import the B-roll:** File → Import Media → select all 12 clips → import. They appear in the browser at the top.
6. **Drag clips onto the timeline** above the audio track, in order, matching the time ranges in the shot list. To trim: hover the edge of a clip and drag inward.
7. **Mute B-roll original audio:** for each clip, right-click → **Detach Audio** → select the detached audio strip → delete. (Pexels clips often have unwanted background noise.)
8. **Add a title card at 0:00:** Titles tab (top) → drag "Standard" title onto the very start → text "TRUE REDDIT HORROR STORY" → make it 2 seconds long, then it fades into Scene 1
9. **Add an end card at 3:22:** drag another title at the end → text "SUBSCRIBE FOR PART 2" → 3 seconds, black background
10. **Add background music:** Audio tab (top) → Sound Effects → search "ambient" or "tension" → drag onto a second audio track → drop to about 25% volume (drag the volume line down inside the clip)
11. **Add the horror color look:** select a video clip → click the Color Balance icon (top right of viewer) → choose **Shadow/Highlight** preset (darkens shadows). Then click Color Correction icon → drag Saturation slider down ~25%, Temperature slider slightly blue. Repeat for each clip or use Cmd+C / Cmd+Option+V → Paste Adjustments to copy the look across all clips.
12. **Skip captions in iMovie.** YouTube will auto-generate captions during upload — that's how 90% of Reddit horror channels do it. (Optional: in YouTube Studio after upload, paste in the `story1_narration.srt` file for perfect captions instead of waiting for YT's auto-caption.)
13. **Export:** File → Share → File → Resolution 1080p, Quality High, Compress Faster → Next → save as `story1_final.mp4`

## Optional polish (15 min each, iMovie)

- **Ken Burns zoom on Scene 4 (the door):** select the clip → Crop icon (top of viewer, next to Color) → Ken Burns → set the Start frame to wider and End frame to tighter on the doorknob. Auto-animates the zoom across the clip.
- **Sound effects:** Audio tab → Sound Effects → search "thud" or "low boom" — drop one at 0:46 (the word "door"). Search "breathing" — drop a quiet one at 2:36 (under the "Breathing. Slow. Patient." beat).
- **Slow-motion the centerpiece:** select the Scene 4 door clip → Speed icon → Slow → 50%. Stretches it for tension. Re-trim audio sync afterward.
- **Letterbox bars (cinematic look):** add a Title at the top with two black bars (use the "Black" background option), set duration to full video. Optional but makes it feel like a horror trailer.
- **YouTube post-upload captions:** in YouTube Studio after upload → Subtitles → Add → Upload File → Without Timing → choose `story1_narration.srt`. Saves you from YouTube's sometimes-wrong auto-captions.
